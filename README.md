# 《龙珠：厄日序曲》小说工程库

> 龙珠多元宇宙末日挽歌 | 作者：七枫 | 指导：摩伦

## 项目概述

- **类型**：龙珠同人小说
- **基调**：悲壮 / 宿命 / 绝望
- **目标字数**：35-45万字
- **结构**：四卷制

## 目录结构

```
龙珠厄日序曲/
├── 00_核心文档/     # Claude优先加载（创作圣经、大纲、进度蒸馏）
├── 01_世界观/       # 静态设定（术语、能力边界、宇宙战况）
├── 02_人物设定/     # 按阵营分组（Z战士组、凡人组、反派组等）
├── 03_正文/         # 按卷分目录（卷一~卷四）
├── 04_创作简报/     # 章节简报存档
├── 05_草稿/         # 临时草稿
├── 06_脚本/         # Python辅助工具
├── 07_输出/         # 合并输出
└── .claude/         # Claude协作配置
```

## Claude Code 协作

### 创作新章节时加载顺序
1. `00_核心文档/创作圣经.md`
2. `00_核心文档/附录_进度蒸馏.md`
3. `03_正文/[当前卷]/_卷X大纲.md`
4. `03_正文/[当前卷]/[上一章].md`
5. `02_人物设定/[本章POV角色].md`

### 辅助脚本
- `anti_ai_checker.py` - 反AI味句式检测
- `pov_validator.py` - POV越界检测
- `progress_tracker.py` - 进度统计
- `briefing_generator.py` - 简报生成

## 核心原则

1. **终局不可逆** - Z战士全员战死，多元宇宙崩坏
2. **人设优先于剧情** - 冲突时修改剧情而不是歪曲角色
3. **人味 > 工整** - 优先牺牲工整，保留人味
4. **反AI味** - 禁用模板句式，用身体与环境说话
