# Claude Code 协作工作流

> 《龙珠：厄日序曲》创作协作标准流程

---

## 标准创作流程

```
1. 用户提供章节需求（一两句话）
   ↓
2. Claude读取必要上下文（见 context_loader.md）
   ↓
3. Claude生成章节创作简报初稿
   ↓
4. 用户审核/修改简报
   ↓
5. Claude按简报生成章节草稿
   ↓
6. 运行自检脚本
   ↓
7. Claude根据自检结果修改
   ↓
8. 用户最终审核
   ↓
9. 更新进度蒸馏文档
```

## 八条硬规则（必须遵守）

1. **终局不可逆** - 禁止逆转Z战士全灭结局
2. **POV锁定协议** - 角色不知道的东西，不写成事实
3. **人设优先于剧情** - 冲突时修改剧情而不是歪曲角色
4. **情绪不解释** - 用身体与环境说话
5. **不写总结性评语** - 避免说教式旁白
6. **反AI味句式黑名单** - 禁用"没有XX，只有XX"等句式
7. **移动端排版** - 3-4行以上段落必须拆
8. **章节结尾钩子** - 必须有"钩子"或"象征性画面"

---

## 反AI味句式黑名单

禁止使用：
- "没有XX，没有XX。"
- "没有XX，只有XX。"
- "不是XX，而是XX。"
- "仿佛XX一般/一样"（限量使用）
- "世界仿佛被按下了XX键"

## 自检清单

生成章节后，请检查：

- [ ] 是否有视角越界？
- [ ] 是否出现解释型总结句？
- [ ] 是否使用了黑名单句式？
- [ ] 段落是否超过3-4行？
- [ ] 章节结尾是否有钩子？
- [ ] 人设是否被歪曲？
