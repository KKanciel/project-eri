# Claude Code 上下文加载指南

> 本文件指导Claude Code在协作创作时如何加载必要的上下文

---

## 创作新章节

请依次读取以下文件：

1. **创作圣经**（必须）
   ```
   00_核心文档/创作圣经.md
   ```

2. **进度蒸馏**（必须）
   ```
   00_核心文档/附录_进度蒸馏.md
   ```

3. **当前卷大纲**（必须）
   ```
   03_正文/[当前卷]/_卷X大纲.md
   ```

4. **上一章内容**（必须）
   ```
   03_正文/[当前卷]/[上一章].md
   ```

5. **本章POV角色设定**（按需）
   ```
   02_人物设定/[阵营]/[角色].md
   ```

---

## 修改已有章节

请依次读取以下文件：

1. **创作圣经**
   ```
   00_核心文档/创作圣经.md
   ```

2. **目标章节文件**
   ```
   03_正文/[卷]/[章节].md
   ```

3. **相关角色设定**（按需）
   ```
   02_人物设定/[阵营]/[角色].md
   ```

## 优先级顺序

1. **创作圣经** - 文风/POV/反AI味（最高优先级）
2. **附录_进度蒸馏** - 剧情进度&硬设定&时间线
3. **人物设定** - 人设本质&句式指纹
4. **故事总大纲** - "发生什么"+"大致怎么排布"
